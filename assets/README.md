# Baskar's Portfolio Assets

This directory is for storing images, icons, and other media files.

## Usage

Add the following types of files here:

- **Profile Images**: profile.jpg, avatar.png
- **Project Images**: project1.jpg, project2.jpg
- **Icons**: custom icons if not using Font Awesome
- **Documents**: Resume PDF, certificates
- **Other Media**: Videos, gifs, etc.

## Example Reference in HTML

```html
<!-- Profile Image -->
<img src="assets/profile.jpg" alt="Baskar">

<!-- Project Image -->
<img src="assets/projects/eap-software.jpg" alt="Textile ERP Software">

<!-- Document -->
<a href="assets/documents/resume.pdf" download>Download Resume</a>
```

## Optimization Tips

1. **Compress Images**: Use tools like TinyPNG or Compressor.io
2. **Use Modern Formats**: WebP for better compression
3. **Responsive Images**: Use `srcset` for different screen sizes
4. **Lazy Loading**: Use `loading="lazy"` on images
